# 📝 Advanced Todo App

A fully responsive, feature-rich todo application built with React featuring local storage persistence, real-time filtering, and a modern UI.

## ✨ Features

- ✅ Add, delete, and complete todos
- 🔄 Real-time filter: All, Active, Completed  
- 💾 Persistent storage using Browser's localStorage
- 📱 Fully responsive design (mobile & desktop)
- ⚡ Smooth animations and transitions
- 🎯 Beautiful gradient UI

## 🛠️ Tech Stack

- **React.js** - UI library
- **React Hooks** - useState, useEffect for state management
- **localStorage** - Client-side data persistence
- **CSS3** - Flexbox, gradients, animations

## 📦 Installation & Setup

```bash
git clone https://github.com/samantaray-kalyani/todo-app.git
cd todo-app
npm install
npm start
```

## 💡 Key Concepts Demonstrated

- React Hooks (useState, useEffect)
- Component-based architecture
- localStorage API
- Event handling
- Conditional rendering
- Array methods (filter, map)
